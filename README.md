# Interactive Squeaky Toy PCB Design

## Overview
This repository contains the PCB design of an Interactive Squeaky Toy developed using KiCad. The project includes the schematic design, PCB layout, 3D PCB views, and Gerber files required for PCB manufacturing.

## Design Tool
- KiCad

## Project Files
- Schematic  (PDF)
- PCB_Layout Image
- 3D_Top_View
- 3D_Bottom_View
- Gerber Files (ZIP)

## Features
- Compact PCB Layout
- Proper Component Placement
- PCB Routing
- 3D Visualization
- Manufacturing-Ready Gerber Files

## Applications
- Interactive Electronic Toys
- Embedded Systems Projects
- PCB Design Learning
- Engineering Portfolio Projects
